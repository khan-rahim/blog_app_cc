About

Lets Blog is a blogging website which is build using django as backend, containerized on docker and hosted on AWS S3 to provide container as a service.
Features :-
  • Create, Update and Delete posts.
  • Login, Logout.
  • Create and Update profile, Set profile picture.
  • Password reset using Email.
